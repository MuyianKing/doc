hl-form-item 继承了el-form-item所有的属性，在 el-form-item 基础上 hl-form-item 预设了一些校验规则

基础用法 ​

姓名
年龄
身份证号
手机号
vue

上面的 required 表示姓名为必填项，当没有填写时会提示“请输入姓名”

规则属性名说明
必填required-
身份证号idCard-
邮箱email-
整型integer-

手机号phone-

最大长度maxLength绑定一个数字
最小长度minLength绑定一个数字

属性说明类型可选值默认值
